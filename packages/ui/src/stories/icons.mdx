## 사용 방법

### Installation

```bash
yarn add @zagdang/icons
```

### Props

<table>
  <thead>
    <tr>
      <th>Prop</th>
      <th>Type</th>
      <th>Default</th>
      <th>Description</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>size</td>
      <td>number</td>
      <td>24</td>
      <td>아이콘 크기</td>
    </tr>
    <tr>
      <td>color</td>
      <td>string</td>
      <td>currentColor</td>
      <td>아이콘 색상</td>
    </tr>
    <tr>
      <td>strokeWidth</td>
      <td>number</td>
      <td>2</td>
      <td>선 굵기</td>
    </tr>
  </tbody>
</table>

### Example (color, size 변수 사용)

```tsx
import { DashBoard } from '@zagdang/icons';

const Example = () => (
  <DashBoard
    size={48}
    color="primary"
    strokeWidth={2}
  />
);
```
